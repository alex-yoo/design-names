# DesignNames Project

참고사이트 링크 : https://koreanname.me/
전자가족관계시스템 : http://efamily.scourt.go.kr/

# 기능
1. 연도별 이름 순위
2. 이름 찾기
    - 제외 할 글자
    - 포함 할 글자
    - 성
3. 이름 길이별 순위
4. 개명 이름 순위
5. 출생아 통계
6. 성과 이름 부정 조합 단어
7. 이름 별 리뷰
8. 회원 가입
    - 구글
    - 네이버
    - 카카오
    - 페이스북
9. 회원 별 검색기록
10. 찜하기
